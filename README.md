Veri Yapıları ve Algoritmalar konusunda yaptığım ilk proje.

> ## Soru 1
[22,27,16,2,18,6] -> Insertion Sort türüne göre aşamalarını yazınız.

> ## Cevap 1
```
1. aşama : [2, 27, 16, 22, 18, 6]
2. aşama : [2, 6, 16, 22, 18, 27]
3. aşama : [2, 6, 16, 18, 22, 27]
